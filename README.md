Project created to understand how can use GraphQL in a simple CRUD.

In this project was used [gqlgen](https://github.com/99designs/gqlgen)

## Install

```sh
go build 
```

## Usage

- Start mongoDb:
```sh
./docker-compose run
```

- Start the project:
```sh
go run server.go
```
